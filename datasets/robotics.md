# Robotics Datasets

Sensor data, maps, and logs from our robotics research.

## Available Data

### SLAM Maps
- Indoor environment maps
- Outdoor terrain maps
- Multi-floor buildings

### Sensor Logs
- LIDAR point clouds
- Camera feeds (anonymized)
- IMU and odometry data

### Autonomous Vehicle Logs
- Driving scenarios
- Obstacle avoidance events
- Path planning data
